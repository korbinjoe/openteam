Readable titles, always

Derived from the first user message the moment it lands. Locked the instant you rename. Placeholders never reach the eye.

Replaces six identical New Task rows that forced the operator to remember open-order.

Workspace headers as separators

10 px uppercase tracking-wide muted type. The list breathes; the active workspace stays anchored without dominating.

Multiple workspaces stay on screen permanently — headers must not compete with task rows.

Hover adds, never replaces

Timestamps and agent counts stay visible. Pin · archive · add-agent layer in on the right via opacity, no horizontal jitter.

The user often hovers because they want to read the timestamp.

Two-segment workload count

3·12 = three active, twelve total. The first number is the one you scan; the second sits faintly beside it.

Replaces a single number that read as either inbox count or unread badge — neither was right.

Spacing rhythm

Sidebar uses 4 / 8 / 12 / 16 / 24. Task rows sit on a 30 px baseline, agents on 24 px. Workspaces breathe with a 20 px gap above each header.
